# Projetos

Projetos integradores multiarquivo e as soluções dos exercícios
guiados. Cada projeto roda a partir do próprio diretório (em geral
`lua5.4 principal.lua`) e carrega asserts próprios, executados pela
suíte de testes como qualquer lição.

| Pasta | O que contém |
|-------|--------------|
| `exercicios/` | Soluções dos exercícios propostos em `documentacao/comece_aqui.md`, um arquivo por exercício |
| `pluralizador/` | Plural de substantivos do inglês: cadeia irregulares → exceções → regras, com rockspec validado na CI |
| `json/` | Codificador/decodificador de JSON sem bibliotecas externas, com validação estrita |
| `equipe/` | Agentes com ferramentas: um programa estruturado em módulos, inspirado em arcabouços de agentes |

`pluralizador/`, `json/` e `equipe/` têm README próprio com os detalhes
e as limitações de cada projeto.
