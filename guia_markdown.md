# Guia Básico de Markdown

Markdown é uma linguagem de marcação leve criada em 2004 por John Gruber, com o objetivo de permitir que as pessoas "escrevam usando um formato de texto fácil de ler e fácil de escrever, que possa ser convertido de forma limpa em HTML". É amplamente utilizado em plataformas como GitHub, Reddit e em sistemas de documentação devido à sua simplicidade e eficiência. Este guia cobre os fundamentos do Markdown, ideal para quem está começando.

## Por Que Usar Markdown?

Além de sua simplicidade e eficiência, o Markdown é crucial para escritores de conteúdo técnico, programadores e qualquer pessoa que busque uma maneira rápida e fácil de criar páginas web. Ele simplifica o processo de formatação, é fácil de aprender e pode ser convertido em HTML automaticamente, tornando-se uma habilidade essencial no mundo digital atual.

## Cabeçalhos

Use `#` para criar cabeçalhos. O número de `#` determina o nível do cabeçalho.

```markdown
# Cabeçalho nível 1
## Cabeçalho nível 2
### Cabeçalho nível 3
#### Cabeçalho nível 4
##### Cabeçalho nível 5
###### Cabeçalho nível 6
```

## Ênfase

- **Negrito**: Envolver o texto com `**` ou `__`.

  `**texto em negrito**` ou `__texto em negrito__`

- _Itálico_: Envolver o texto com `*` ou `_`.

  `*texto em itálico*` ou `_texto em itálico_`

- ~~Tachado~~: Envolver o texto com `~~`.

  `~~texto tachado~~`

## Listas

### Não Ordenadas

Use `*`, `+`, ou `-`.

```markdown
- Item 1
- Item 2
  - Subitem 2.1
  - Subitem 2.2
```

### Ordenadas

Use números seguidos de um ponto.

```markdown
1. Primeiro Item
2. Segundo Item
   1. Subitem 2.1
   2. Subitem 2.2
```

## Links

Crie links facilmente:

```markdown
[Texto do link](URL)
```

Exemplo: `[Google](https://www.google.com)`

## Imagens

Incorpore imagens assim:

```markdown
![Texto alternativo](URL_da_imagem)
```

## Código

Para código inline, use \` (crase).

`` `código inline` ``

Para blocos de código, use três crases ou indentação com 4 espaços.

````markdown
    ```
    Bloco de código
    ```
````

## Citações

Destaque citações:

```markdown
> Isto é uma citação.
```

## Linhas Horizontais

Crie uma separação visual:

```markdown
---
```

## Tabelas

Organize informações em tabelas:

```markdown
| Cabeçalho 1 | Cabeçalho 2 |
| ----------- | ----------- |
| Conteúdo 1  | Conteúdo 2  |
| Conteúdo 3  | Conteúdo 4  |
```

## Exemplo Completo

Veja como tudo se junta em um exemplo:

```markdown
# Guia de Markdown

Markdown é uma linguagem **simples** e _eficaz_ para formatação de texto.

## Principais Elementos

- Listas
- Links
- Código
- Tabelas

Visite [OpenAI](https://www.openai.com) para mais informações.
```
